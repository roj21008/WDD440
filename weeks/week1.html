What is the DOM?

The Document Object Model is a programming interface that allows us to create, change, or remove objects from the document. We can also add events to the document to make the page more dynamic.

What is the difference between Imperative vs Declarative Programming - Think about the difference between step-by-step instructions on how to make a pizza vs just ordering a pizza.

Imperative programming

Involves giving the computer step-by-step instructions on how to do something. For instance, if you were to tell a computer how to make a pizza, you'd need to describe each step: making the dough, adding sauce, spreading toppings, and baking.

Declarative programming

It tells the computer what the end goal is without specifying each step. Ordering a pizza is declarative—you say, "I want a pepperoni pizza," without detailing how to make it. The same idea applies in code: instead of specifying each step, you declare the result, and the program handles the details.

In React, a popular library for building interfaces, we write code declaratively. We describe what we want the interface to look like, and React figures out how to make it happen in the DOM.

What is JSX - the syntax extension for JavaScript?

JSX (JavaScript XML) is a sintax that allows to write HTML like a code within JS. It’s used heavily in React to define what the UI should look like.
